# Remote Jobs

## Conventions

- Write code, comments, and commits in English
- Use single quotes for JS/TS/Astro frontmatter strings
- Keep `.astro/`, `.git/`, `node_modules/`, `dist/` untouched
- Never delete or recreate `.astro/` — Astro's dev server owns it

## Style

- Match existing patterns in the codebase
- Use Tailwind utility classes for styling
